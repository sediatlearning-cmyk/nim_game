# Jeu des allumettes

## Description

Le jeu commence avec **21 allumettes disposées dans un seul tas**.

Deux joueurs s'affrontent à tour de rôle. L'objectif est de ne pas être le joueur qui retire la dernière allumette.

## Déroulement de la partie

Au lancement du jeu, les informations suivantes sont demandées :

- Le nom du premier joueur ;
- Le nom du deuxième joueur ;
- Le joueur qui commencera la partie.

## Règles du jeu

- Les joueurs jouent chacun leur tour.
- À chaque tour, le joueur doit retirer **entre 1 et 4 allumettes**.
- Un joueur ne peut pas retirer plus d'allumettes qu'il n'en reste dans le tas.
- La partie continue jusqu'à ce qu'il ne reste plus aucune allumette.

## Condition de victoire

Le joueur qui retire **la dernière allumette perd la partie**.

L'autre joueur est alors déclaré **gagnant**.
